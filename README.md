# RSSFeeder


### Step 1: Set up

Set up the url which you will be fetching from within `settings/inputSettings.php`

```php

<?php

return [

    /*
    |-----------------------------------------------------------------
    | Default Url
    |-----------------------------------------------------------------
    |
    | You can set up the Url
    |
    */

    'url' => '',

];

```


### Step 2: Run

* With Console
```php
php index.php
```

* With Browser
```php
php -S localhost:8000 index.php
```
